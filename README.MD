# Draeyes 兽装可动眼系列 手动控制遥控器使用说明书

## 概述

**Draeyes 兽装可动眼系列 手动控制遥控器** 是麟云电科开发的兽装控制设备，适用于 **Draeyes 兽装可动眼系列1.0-3.0版本**。控制器集成了手动控制、完全自动模式和智能眨眼功能，能够提供自然、流畅的眼球运动控制体验。该控制器采用ESP-NOW低延迟无线协议，确保实时响应，支持室内50米范围内稳定连接。

---

## 产品特点

| 特性 | 描述 |
|------|------|
| **多模式控制** | 支持手动、完全自动、智能眨眼三种控制模式 |
| **低延迟通信** | ESP-NOW协议，响应时间小于20毫秒 |
| **智能算法** | 内置仿生眼球运动算法，模拟真实眼球行为 |

| **即插即用** | 接受端设备开机后自动记忆配对信息直到当前开机生命周期结束 |

## 控制器布局

### 按键分布图
```
        [摇杆]
    [R1]      [R2]
    
右侧边缘（从上到下）:
             [LB]  ← 左眼控制
             [BK]  ← 同步开关
             [RB]  ← 右眼控制
```

### 按键功能说明
| 按键 | 功能描述 |
|------|----------|
| **摇杆** | 控制眼球XY轴移动方向 |
| **R1** | 模式切换和摇杆锁定功能 |
| **R2** | 系统功能和预留操作 |
| **LB** | 左眼眨眼控制（手动模式下） |
| **RB** | 右眼眨眼控制（手动模式下） |
| **BK** | 双眼同步开关 |

## 控制模式详解

### 模式一：手动模式（默认）
> **适用场景**: 需要精确控制眼球动作的展示场合

#### 控制方式
- **眼球移动**: 使用摇杆精确控制XY方向
- **左眼眨眼**: LB按键控制（按下闭眼，松开睁眼）
- **右眼眨眼**: RB按键控制（按下闭眼，松开睁眼）
- **双眼同步**: BK开关开启时，右眼自动跟随左眼动作

#### 动画效果参数
| 触发方式 | 闭眼步数 | 睁眼步数 | 步长间隔 |
|----------|----------|----------|----------|
| **LB触发** | 6-8步渐变 | 7-9步渐变 | 10毫秒 |
| **RB触发** | 3-5步渐变 | 7-9步渐变 | 10毫秒 |

### 模式二：完全自动模式
> **适用场景**: 长时间展示、无人值守或需要自然眼神效果的场合

#### 运动算法
| 运动类型 | 移动范围 | 持续时间 | 特点 |
|----------|----------|----------|------|
| **大眼跳运动** | 半径60像素圆形区域 | 83-167毫秒 | 模拟大范围扫视 |
| **微眼跳运动** | 半径8像素小范围 | 7-26毫秒 | 细微调整，更频繁 |
| **凝视停留** | - | 最长3秒 | 随机停留时间 |

#### 智能眨眼系统
- **随机频率**: 每2-6秒随机触发一次眨眼
- **自然节奏**: 先快速闭眼（50-101毫秒），再缓慢睁眼（闭眼时间×2）
- **智能避让**: 在眼球运动过程中智能调整眨眼时机

### 模式三：智能眨眼模式
> **适用场景**: 需要手动控制视线方向，但希望眨眼自然化的场合

#### 控制特点
- **手动方向**: 使用摇杆精确控制眼球移动方向
- **自动眨眼**: 系统自动处理眨眼动作，无需手动按键
- **最佳结合**: 兼具手动控制的精确性和自动眨眼的自然性

## 详细操作指南

### 快速上手

#### 设备配对
1. **配对操作**: 
   ```
   R1 + R2 同时长按 3秒
   ```
2. **配对确认**: 尝试控制，当接收端设备响应时配对成功

### 核心操作

#### 模式切换
```
R1 连续快按 3次（间隔不超过1秒）
手动模式 → 完全自动模式 → 智能眨眼模式 → 手动模式（循环）
```

#### 摇杆锁定
```
R1 长按 3秒
```
- **生效条件**: 仅在手动模式和智能眨眼模式下有效
- **锁定状态**: 摇杆输入被忽略，眼球保持中心位置
- **使用场景**: 临时固定视线方向，便于进行其他操作

#### 摇杆校准
```
R1 + R2 + LB + RB 四键同时长按 3秒
```
> **重要**: 确保摇杆处于物理中心位置再进行校准

#### 其他功能
```
R2 连续快按 3次 → 向接收端发送眼皮切换请求
```

## 手动模式详细操作

### 摇杆控制技巧

| 操作 | 说明 |
|------|------|
| **基础移动** | 轻推摇杆控制移动方向 |
| **精确定位** | 小幅度调整实现精确定位 |
| **平滑控制** | 系统自动5点采样平均，消除抖动 |
| **响应范围** | 全360度控制，支持任意方向移动 |

### 眨眼动作控制

#### 左眼控制（LB）
- **按下效果**: 触发左眼闭合动画（1.0→0.0值变化）
- **松开效果**: 触发左眼睁开动画（0.0→1.0值变化）
- **动画时长**: 闭眼约60-80毫秒，睁眼约70-90毫秒
- **应用场景**: 单眼眨眼表情，左眼独立控制

#### 右眼控制（RB）
- **按下效果**: 触发右眼闭合动画
- **松开效果**: 触发右眼睁开动画
- **动画时长**: 闭眼约30-50毫秒，睁眼约70-90毫秒
- **设计特点**: 右眼闭眼速度更快，适合眨眼辅助

### 同步控制功能
| BK开关状态 | 效果 | 优先级 |
|------------|------|--------|
| **开启** | 右眼自动跟随左眼动作 | RB按键失效 |
| **关闭** | 左右眼独立控制 | 可实时切换 |

## 自动模式运行机制

### 运动轨迹算法详解

#### 大眼跳特征
- **触发条件**: 静止时间超过设定阈值
- **运动范围**: 最大半径60像素的圆形区域
- **运动轨迹**: 随机方向，符合圆形约束
- **运动时间**: 83-167毫秒随机变化
- **停留间隔**: 完成后进入随机时长的凝视状态

#### 微眼跳特征
- **触发条件**: 在凝视过程中的细微调整
- **运动范围**: 最大半径8像素的小范围调整
- **运动轨迹**: 基于当前位置的微小偏移
- **运动时间**: 7-26毫秒快速调整
- **频率特点**: 比大眼跳更加频繁

### 智能眨眼算法详解

#### 频率控制机制
| 参数 | 数值 | 说明 |
|------|------|------|
| **基础频率** | 2-6秒随机间隔 | 模拟正常眨眼频率 |
| **动态调整** | 根据眼球运动状态 | 智能调整频率 |
| **避让机制** | 运动期间降低频率 | 避免冲突 |

#### 动作模拟特性
- **闭眼阶段**: 快速下降，模拟眼睑快速闭合
- **睁眼阶段**: 缓慢上升，模拟自然睁眼过程
- **时间比例**: 闭眼时间：睁眼时间 = 1：2
- **随机变化**: 每次眨眼的具体时长都有微小变化

## 故障排除与维护

### 常见问题诊断

#### 连接问题
> **现象**: 控制指令无响应或延迟严重

**排查步骤**:
1. 检查控制器和接收端的电源状态
2. 确认设备间距离不超过有效范围
3. 检查是否有强干扰源（微波炉、其他2.4GHz设备）
4. 重新执行设备配对流程
5. 如问题持续，重启两端设备

**解决方案**:
- 保持设备间距离在30米以内获得最佳性能
- 避免在WiFi密集区域使用

#### 摇杆问题
> **现象**: 摇杆回中后眼球位置偏移

**解决步骤**:
1. 将摇杆手动调整到物理中心位置
2. 同时按住 `R1+R2+LB+RB` 四个键
3. 保持按压状态3秒钟
4. 看到串口提示"校准完成"后松开按键
5. 测试摇杆回中是否正常

> **现象**: 摇杆响应不灵敏或有死区

**解决方案**:
- 检查摇杆物理连接是否松动
- 使用压缩空气清理摇杆底座灰尘
- 如问题严重，联系售后更换摇杆模块

### 维护保养

#### 日常保养
- **清洁**: 使用干布或微湿布清洁外壳，避免液体进入按键缝隙
- **存放**: 放置在干燥、避光的环境中
- **充电**: 定期充电，避免长时间深度放电
- **检查**: 定期检查按键手感和摇杆精度

#### 长期维护
- **固件更新**: 定期检查并更新固件版本
- **校准**: 每月进行一次摇杆校准
- **清理**: 每季度进行一次深度清洁
