# Draeyes 兽装可动眼系列 手动控制遥控器使用说明书

## 概述

**Draeyes 兽装可动眼系列 手动控制遥控器** 是麟云电科开发的兽装控制设备，适用于 **Draeyes 兽装可动眼系列1.0-3.0版本**。控制器集成了手动控制、完全自动模式、智能眨眼以及**全新的智能感光调光系统**。该控制器采用ESP-NOW低延迟无线协议，确保实时响应，支持室内50米范围内稳定连接。

## 控制器布局

### 按键分布图

```
        [摇杆]
    [R1]      [R2]
    
右侧边缘（从上到下）:
             [LB]  ← 左眼控制 / 亮度增加(+)
             [BK]  ← 同步开关
             [RB]  ← 右眼控制 / 亮度减少(-)
```

### 按键功能说明

| 按键 | 常规模式功能 | **调光配置模式功能 (新)** |
|:---:|---|---|
| **R1** | 短按3次：切换运动模式<br>**长按3秒：进入/切换调光子模式** | **长按3秒：切换子模式/保存退出** |
| **R2** | 短按3次：发送眼皮切换请求<br>**长按3秒：摇杆锁定/解锁** | **单击：切换跟随状态 / 执行采样锁定** |
| **LB** | 左眼眨眼控制 | **亮度补偿微调 (+)** (增加亮度) |
| **RB** | 右眼眨眼控制 | **亮度补偿微调 (-)** (减少亮度) |
| **BK** | 双眼同步开关 | - |
| **摇杆**| XY轴移动控制 | - |

> **注意**：如果未检测到 VEML7700 光照传感器，调光配置模式将被禁用，长按 R1 无响应，且系统默认发送亮度值 1.0。

## 核心功能详解

### 1\. 运动控制模式

*(通过 R1 连续快按 3次 循环切换)*

  * **手动模式（默认）**: 摇杆控制视线，LB/RB 控制眨眼。
  * **完全自动模式**: 模拟生物眼球的大眼跳、微眼跳和随机眨眼。
  * **智能眨眼模式**: 手动控制视线方向，系统接管眨眼动作（自然频率）。

### 2\. 智能感光与调光系统 (新功能)

*(通过 R1 长按 3秒 进入，仅在传感器在位时可用)*

该系统允许眼睛根据环境光线自动调整瞳孔大小或背光亮度。调光模式具有记忆功能，退出后设置依然生效。

#### 进入流程与子模式

长按 R1 键，控制器将在以下状态间循环：
`主模式` → `调光子模式1 (跟随设置)` → `调光子模式2 (采样锁定)` → `主模式 (退出并保存)`

#### 子模式详解

| 模式 | 状态指示 | R2 单击功能 | LB / RB 功能 |
|---|---|---|---|
| **子模式1**<br>(跟随设置) | 串口提示<br>"设置跟随" | **切换自动跟随开关**<br>• 开启：亮度随环境光实时变化<br>• 关闭：亮度强制锁定为 1.0 (全亮) | **微调 Offset**<br>LB: 增加补偿<br>RB: 减少补偿 |
| **子模式2**<br>(采样锁定) | 串口提示<br>"采样锁定" | **采样并锁定**<br>读取当前环境光值并永久锁定，同时自动关闭跟随模式。 | **微调 Offset**<br>LB: 增加补偿<br>RB: 减少补偿 |

#### 亮度补偿微调 (Offset)

在调光模式下，如果觉得当前的自动亮度偏暗或偏亮，可通过肩键微调：

  * **LB (Left Button)**: 增加亮度补偿 (+0.05步进)
  * **RB (Right Button)**: 减少亮度补偿 (-0.05步进)
  * *提示：微调值会自动保存到 EEPROM，断电不丢失。*

## 详细操作指南

### 快速上手

#### 1\. 设备配对

  - **操作**: `R1 + R2` 同时长按 3秒
  - **反馈**: 接收端设备响应控制即为成功

#### 2\. 摇杆锁定 (更新)

  - **操作**: `R2` 长按 3秒
  - **适用**: 手动模式、智能眨眼模式
  - **效果**: 锁定当前摇杆输入，再次长按解锁。*(注：旧版本为R1长按，现已更改)*

#### 3\. 摇杆校准

  - **操作**: `R1 + R2 + LB + RB` 四键同时长按 3秒
  - **注意**: 校准时请确保摇杆处于物理中心位置，不要触碰摇杆。

### 光感控制实战

**场景 A：我希望眼睛亮度/瞳孔随环境光自动变化**

1.  确保已安装光感模块。
2.  长按 `R1` 3秒进入 **调光子模式1**。
3.  单击 `R2`，确保串口/监视器提示 "跟随已开启"。
4.  长按 `R1` 两次（经过子模式2 -\> 退出），保存并返回主模式。

**场景 B：现场灯光太乱，我想固定在这个亮度**

1.  站在希望记录的灯光环境下。
2.  长按 `R1` 两次，进入 **调光子模式2**。
3.  单击 `R2`，系统将读取当前光照并锁定该数值。
4.  长按 `R1` 退出。此时无论走到哪里，亮度/瞳孔都将保持该状态。

**场景 C：自动感光太灵敏/太暗了**

1.  进入任意调光子模式。
2.  按压 `LB` (加亮) 或 `RB` (变暗) 进行修正。
3.  观察眼睛反应直到满意，长按 `R1` 退出保存。

## 故障排除

### 调光功能无法使用

> **现象**: 长按 R1 3秒后无法进入调光模式，无串口打印。

**原因**:
系统启动时未检测到 VEML7700 传感器。出于安全考虑，固件自动禁用了相关功能。
**解决**:

1.  检查传感器 I2C 接线 (SDA/SCL) 是否松动。
2.  重启控制器，观察串口是否打印 "Sensor found"。

### 摇杆回中后眼球偏移

**解决**:
执行四键长按校准程序（`R1+R2+LB+RB` 长按3秒）。
